Iniciando o projeto
```shell
npm install grunt-grunticon --save-dev
cd $dir_projeto
npm install
```
Durando o desenvolvimento
```shell
grunt watch

```

Rodanto tests
```shell
grunt jasmine

```

Validando codigo
```shell
grunt jshint

```
